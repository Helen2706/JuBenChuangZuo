这是在潘婵师姐原始代码的基础上进行了增加登录和注册功能
改动如下：
1.增加用户蓝本，用于操作与登录注册相关的功能
2.增加用户表单类和模板类，并且只使用最基本的bootstrap而不是flask-bootstrap渲染表单
3.更改基本的跳转功能，使用户先登录再进入系统。
4.数据库中存储密码散列值
5.当用户注册时会给用户的注册邮箱发送确认邮件，当用户登录邮件并确认时，会检查确认是否成功。
6.添加请求钩子，当用户想要访问某些需要认证后才能访问的页面时会跳转到位认证页面，这使用户可以通过点击指定的链接再次请求服务器发送确认邮件
7.自己创建一个小系统，添加主页面，个人资料页面，满足用户修改个人资料的需求
8.将用户进行分类，像教务系统一样可以选择：编剧、演员、导演、团队、团队中的个人
